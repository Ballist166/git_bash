1) Предварительно создайте файл bash1.txt, в который вы будете добавлять выполненные команды   -> touch bash1.txt
2) Открыть домашнюю директорию через терминал                                                  -> cd
3) Определить имя папки, в которой вы находитесь                                               -> pwd
4) Создать внутри этой папки каталог с именем test1                                            -> mkdir test1
5) Перейти в папку test1                                                                       -> cd test1
6) Создать файл 1,2 и 3 внутри каталога test1                                                  -> touch 1 2 3
7) Проверить содержимое каталога test1                                                         -> ls
8) Перейти в домашнюю директорию                                                               -> cd
9) Создать папку test2 внутри домашней директории                                              -> mkdir test2
10)Удалить папку test2                                                                         -> rmdir test2
11)Удалить файл 2 из папки test1                                                               -> rm test1/2
12)Создать папку в домашней директории test3                                                   -> mkdir test3
13)Добавить в test3 два файла                                                                  -> touch test3/one.txt test3/two.txt
14)Удалить папку test3                                                                         -> rm -rf test3
15)Создать папку test4 в домашней директории                                                   -> mkdir test4
16)Переместить файлы 1 и 3 из папки test1 в папку test4                                        -> mv test1/1 test1/3 test4
17)Добавить в файл 1 три строки со словами line                                                -> echo line > test4/1
                                                                                               -> echo line >> test4/1
                                                                                               -> echo line >> test4/1
18)Посмотреть содержимое файла 1                                                               -> cat test4/1
19)Добавьте в файл 3 три строки со словами line                                                -> printf "line\nline\nline\n" >> test4/3
20)Просмотрите содержимое двух файлов (1 и 3) сразу                                            -> cat test4/1 test4/3
21)Используя один из редакторов замените все строки в файле 1                                  -> nano test4/1
